# SportsHub 🏀⚽🏏

SportsHub is a Flask-based sports equipment e-commerce platform.

## Features

- User Authentication
- Product Catalog
- Shopping Cart
- Checkout
- Order Management
- Admin Dashboard
- Responsive Bootstrap UI

## Tech Stack

- Python
- Flask
- SQLAlchemy
- SQLite
- Bootstrap
- Flask-Login
- Flask-Migrate

## Installation

```bash
git clone https://github.com/nikhilghotankar-8881/SportsHub.git

cd SportsHub

python -m venv venv

pip install -r requirements.txt

python run.py
